# repos

데이터 조회와 집계 보조 함수를 담당합니다.

## 현재 파일

- `room.repo.js`
  - `ORDO_ROOM_DATA`를 읽어 프로젝트, 대기 조치, 리포트 값을 조회합니다.
  - 대시보드, 프로젝트, 승인 화면에서 쓰는 집계, 정렬, 상태 판단 값을 만듭니다.
  - 화면 코드는 데이터 배열을 직접 뒤지지 않고 이 보조 함수를 호출합니다.

지금은 목업 데이터 기반입니다. 나중에 서버 API나 DB가 생기면 이 폴더부터 교체하면 됩니다.

비유하면 창고 직원입니다. 화면이나 서비스는 "프로젝트 가져와"라고 요청하고, repo가 어디서 가져올지 결정합니다.
