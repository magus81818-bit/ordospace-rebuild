# ui

반복해서 쓰는 작은 화면 부품과 HTML 조립 보조 함수를 담당합니다.

## 현재 파일

- `room.ui.js`
  - 룸 계열 화면에서 반복되는 escape, 배지, KPI 카드, 필터 탭, 룸 헤더, 교차 링크, 래퍼 보조 함수를 만듭니다.
  - 데이터 조회는 `app/repos/`, 화면 구성은 `app/screens/` 또는 `app/main.js`에 맡깁니다.
  - 기존 호출부 호환을 위해 `roomEsc`, `roomBadge`, `roomMetric` 같은 함수 이름은 전역에 유지합니다.

## 다음 후보

- 공통 상태 배지.
- 모듈 카드/승인 카드 조립 함수.
- 모달 본문 보조 함수.
- 타임라인 행 보조 함수.

비유하면 레고 블록입니다. 여러 화면은 이 블록들을 조합해서 만들어집니다.
